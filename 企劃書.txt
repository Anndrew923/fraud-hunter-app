# 「誠信名錄」App 完整企畫書（法律合規版）

## 一、專案概述

### 1.1 專案名稱
**「誠信名錄」- 社會信用檔案管理平台**

### 1.2 專案背景
台灣詐騙案件猖獗，每年造成超過1000億台幣損失，而詐騙犯刑責過輕，犯罪成本極低。同時，欠錢不還的債務人過著舒適生活，債權人卻求助無門。現有的防詐app多為被動防禦，缺乏對詐騙犯和欠款人的威懾力。

### 1.3 專案宗旨
- 建立詐騙犯和欠款人的社會信用檔案
- 透過合法途徑公開相關資訊，達到社會警示作用
- 建立「社會性死亡」機制，提高犯罪和欠款成本
- 促進誠信社會建設

### 1.4 目標用戶
- 一般民眾（防詐查詢）
- 債權人（債務人信用查詢）
- 企業主（員工/合作夥伴信用查詢）
- 法律從業人員（案件資料查詢）

## 二、法律合規策略

### 2.1 資料來源合法性
```
✅ 合法資料來源：
1. 法院判決書
   - 司法院公開資料（免費）
   - 可全文引用並註明出處
   - 需隱藏身分證字號、詳細地址

2. 通緝犯資訊
   - 警政署公開資料（免費）
   - 包含姓名、照片、案由
   - 與警政署資料同步

3. 媒體報導
   - 經查證的公開報導（免費）
   - 僅引用必要部分並註明出處
   - 避免全文轉載

4. 政府公告
   - 各級政府公開資訊（免費）
   - 評估公開目的與App目的關聯性

❌ 禁止資料來源：
1. 用戶投稿的個人資料（除非完全匿名或取得當事人同意）
2. 債權人舉報（僅持有本票、借條不合法）
3. 民眾提供線索（未經當事人同意）
4. 不起訴處分書（除非符合例外條件）
```

### 2.2 個資法合規機制
```
✅ 合規措施：
1. 敏感資訊處理
   - 隱藏身分證字號
   - 隱藏詳細地址（僅顯示縣市）
   - 隱藏其他敏感資訊

2. 資料保存期限
   - 法院判決書：15年（與強制執行時效一致）
   - 通緝犯資料：與警政署同步，通緝解除即移除
   - 用戶投稿：較短保存期限，建立審核機制

3. 申訴更正機制
   - 提供明確申訴管道
   - 7-15個工作天內處理
   - 資料有誤立即更正
   - 有爭議則註記
```

### 2.3 免責聲明
```
免責聲明內容：
1. 資料來源聲明：明確指出資料主要來自公開資訊
2. 資料真實性限制：不保證所有資料絕對正確
3. 免責範圍：對因使用資料造成的損害不承擔責任
4. 用戶行為規範：禁止發表不實、誹謗言論
5. 平台審核權利：有權審核、修改或移除內容
6. 不構成法律意見：提供的資訊不構成法律意見
```
